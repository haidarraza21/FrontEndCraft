Character Entity Reference

  1. Used to display reserved or special character.
  2. Syntex often starts with & and ends with ;.

Exapmleof Character Entity Reference

1. Ampersand (&):
Entity name: &
Entity number: &
Example: Tom & Jerry will display as "Tom & Jerry".


2. Less than (<):
Entity name: <
Entity number: <
Example: if (a < b) will display as "if (a < b)".


3. Greater than (>):
Entity name: >
Entity number: >
Example: if (a > b) will display as "if (a > b)".


4. Quotation mark ("):
Entity name: "
Entity number: "
Example: He said, "Hello" will display as "He said, "Hello"".


5. Apostrophe ('):
Entity name: ' (not universally supported, use ' instead)
Entity number: '
Example: It's a nice day will display as "It's a nice day".


6. Non-breaking space ( ):
Entity name:  
Entity number:  
Example: Hello World will display as "Hello World" (with a non-breaking space).
