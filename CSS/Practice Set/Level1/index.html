Basic of CSS

What is CSS

CSS (Cascading Style Sheets) is a stylesheet language used to control the visual presentation of web pages by applying styles to HTML elements, such as colors, fonts, and layouts. It separates content from design, allowing for easier maintenance and responsive web design, where styles adapt to different devices and screen sizes. CSS works through a cascading system, where multiple styles can be applied and overridden based on their specificity and order.

Selector in CSS

In CSS, a selector is used to target HTML elements on a web page that you want to style. The selector is the first part of a CSS rule and determines which HTML element(s) the style rules will apply to.
